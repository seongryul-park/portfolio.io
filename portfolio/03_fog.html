Project Fog

3D Graphics

Project Overview

Implemented various types of fog effects

  • Linear Fog
  • Exponential Fog
  • FragCoord.z Fog

Tasks Completed

Implemented Linear Fog

  • Vertex Shader : Passed model-to-view coordinates for distance calculation
  • Fragment Shader : Applied linear fog effect based on distance using smoothstep

Implemented Exponential Fog

  • Vertex Shader : Passed model-to-view coordinates for distance calculation
  • Fragment Shader : Applied exponential fog effect using exp2

Implemented FragCoord.z Fog

  • Fragment shader : Applied linear fog effect based on distance using FragCoord.z value and smoothstep

Reflection

Reflect on the project

  • Learned how to implement various types of fog effects using shaders
  • Applied different fog effects such as, linear, exponential, and FragCoord.z

What I learned from the experience

  • Gained a deeper understanding of shader programming
  • Realized that effective visual effects can be achieved with relatively simple shader code
