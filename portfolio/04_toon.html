Project Toon Shading

3D Graphics

Project Overview

Implemented various styles of Toon Shading based on Cel Shading

  • Custom Ranges Toon Shading
  • Uniform Ranges Toon Shading
  • Gradient Texture Toon Shading
  • Applied Anti-Aliasing in all three shading styles

Tasks Completed

Implemented Cel Shading for all toon shading styles

  • Sent the normal matrix as a uniform, calculated from the model and view matrices
  • Vertex Shader: Passed normal vector for diffuse and specular calculations
  • Fragment Shader: Calculated diffuse and specular shading using equations

Implemented Custom Ranges Toon Shading

  • Devided colors using conditional logics
  • Applied Anti-Aliasing for Custom Ranges Toon Shading

Implemented Uniform Ranges Toon Shading

  • Devided colors into uniform chunks using chunkNumber
  • Applied Anti-Aliasing for Uniform Ranges Toon Shading

Implemented Gradient Texture Toon Shading

  • Devided colors using a gradient texture
  • Applied Anti-Aliasing for Gradient Texture Toon Shading

Applied Linear Fog and mixed the calculated Toon Shading Color

  • Vertex Shader : Passed model-to-view coordinates for distance calculation
  • Fragment Shader : Applied linear fog effect based on distance using smoothstep and mixed the calculated color
