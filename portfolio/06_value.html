Project Value Noise

3D Graphics

Project Overview

Implemented value noise using permutation hashing

Tasks Completed

  • Implemented 1D, 2D, and 3D value noise evaluation using linear, bilinear, and trilinear interpolation
  • Implemented permutation hashing for noise computation
  • Implemented smoothing methods : Linear, Cosine, SmoothStep, and Quintic

Reflection

Reflect on the project

  • Realized permutation hasing using bitwise operation(&) is more efficient than modulo calculation
  • Realized value noise has a potential to generate unique textures without using image textures
